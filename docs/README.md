# 快速開始


#### 安裝組件庫

```bash
npm i rainbow520-ui
```



#### 引用組件庫
> 在 main.js 中引用組件庫

```javascript
// 全部引用
import 'rainbow520/dist/css/index.css'
import RUI from 'rainbow520/components/lib/index'
Vue.use(RUI)

//按需引用
import 'rainbow520/dist/css/card.css'
import Card from 'rainbow520/components/lib/index'
Vue.use(Card)
```
